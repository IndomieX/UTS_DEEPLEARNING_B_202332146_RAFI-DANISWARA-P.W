# UTS_DEEPLEARNING_B_202332146_RAFI-DANISWARA-P.W


**Petunjuk Singkat Menjalankan Kode (Google Colab)**

1.  Jalankan file `.ipynb` di Google Colab (bisa pakai `Runtime` -> `Run all`).
2.  **PENTING:** Pastikan `dataset_citra.zip` sudah Anda **ekstrak (unzip)** terlebih dahulu.
3.  Saat program berjalan, kode akan **PAUSE** setelah selesai melatih Model A dan akan **meminta Anda meng-upload file**.
4.  **Unggah 30 file gambar** (hasil ekstrak tadi) sekaligus.
5.  Setelah *upload* selesai, program akan lanjut otomatis.
6.  Tunggu hingga semua proses (evaluasi Model A, *training* Model B, evaluasi Model B) selesai. *Done*.
